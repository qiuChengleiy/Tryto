### 如何使用本项目

* 您可以克隆或者下载本项目
```js
git clone git@github.com:qiuChengleiy/Tryto.git

```

* 引入本项目
```js
<script src="tryto.js"></script>
```

* tip:
```js

不要引入这个文件 因为它是不能直接被浏览器识别的 您应当引入编译后的js文件
<script src="tryto.ts"></script>

```
